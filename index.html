Malware

Definition:

Malware, short for 'Malicious Software' is intrusive software that is intentionally designed to cause disruption, damage, or gain unauthorised access to a computer, server, client or computer network


Examples of Malware

Viruses

A computer virus is a type of computer program that, when executed, replicates itself by modifying other computer programs and inserting its own code. If this replication succeeds, the affected areas are then said to be "infected" with a computer virus

Worms

A standalone malware computer program that replicates itself in order to spread to other computers. Often uses computer network to spread itself, relying on security failures on the target computer to access it

Trojan Horses

Type of malware that downloads onto a computer disguised as a legitimate program. Delivery method typically sees an attacker use social engineering to hide malicious code within legitimate software to try and gain users' system access with their software

White-Hat Vs. Black-Hat

Definitions:

White-Hat Hacking - The use of ethical security hacking. Ethical hacking is a term meant to imply a broader category than just penetration testing, under the owner's consent, white-hat hackers aim to identify any vulnerabilities the current system has

Black-Hat Hacking - the use of unethical security hacking. Black hat hackers break into secure networks and systems with the motive of destroying, modifying, or stealing some sensitive data, or to make the networks unusable for authorized network users. Unlike the white hat hackers, black hat hackers do not have any common and standardized code or internal regulation

Open-Source Intellignece

Definition:

Collection and analysis of data gathered from open sources (covert and publicly available) to produce actionable intelligence. OSINT is used in national security, law enforcement, and business intelligence and is of value to analysts who use non-sensitive intelligence in answering classified, unclassified, or proprietary intelligence requirements across the previous intelligence disciplines
